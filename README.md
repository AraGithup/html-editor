# HTML 编辑器

一个纯前端、无需安装的 HTML 可视化编辑器，直接在浏览器中运行。

## 使用方式

直接打开 `index.html`，或通过 GitHub Pages 访问：
**https://aragithub.github.io/html-editor/**

## 功能

### 上传
- **上传 HTML**：单个 `.html` 文件
- **上传文件夹**：含图片、视频等资源的完整项目，资源自动转为 base64 内嵌
- 支持拖拽上传（文件和文件夹均可）

### 文字编辑
- 直接点击页面任意文字即可编辑
- 工具栏支持：加粗、斜体、下划线、删除线、字号、字体颜色、对齐方式

### 图片 / 视频替换
- 鼠标悬停图片或视频，点击「🔄 替换」按钮上传新文件

### 链接 / 按钮编辑
- **直接点击**：触发跳转（链接导航 / 按钮 onclick）
- **悬停 800ms 后点击**：进入文字编辑模式（出现蓝色虚线框提示）
- 悬停时出现「🔗 编辑链接」按钮，点击可修改 `href` 和 `onclick`

### 背景色编辑
- 工具栏点击「🎨 背景色」开关开启模式
- 开启后，页面中有背景色/渐变的元素右上角出现编辑按钮
- 纯色：color picker + hex 输入
- 渐变：可视化编辑器，支持修改类型（线性/径向）、角度、色标颜色和位置，实时预览

### 多页面支持
- 上传文件夹时自动识别所有 HTML 文件
- 工具栏左右箭头在页面间导航
- 页面内链接点击自动跳转到对应页面

### 保存
- **更新原稿**：以原文件名下载，文件夹项目自动还原相对路径
- **保存副本**：文件名加时间戳；文件夹项目可选择是否将资源内嵌为单文件

## 技术说明

- 纯原生 HTML + JS，无任何依赖
- 编辑在 `<iframe>` 内进行，通过 `postMessage` 与父页面通信
- 资源文件通过 `FileReader` 转为 base64，保存时可选还原为相对路径
